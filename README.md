## 📌 Internship Overview
This repository contains a consolidated collection of projects and security tasks completed during my **Cybersecurity Internship at Thiranex**. The tasks span across secure application development, machine learning for threat detection, and vulnerability assessments.

---

## 🚀 Project Modules 

### 📂 Task 1: Security Automation & Analysis
*   **Description:** Developed Python-based security utilities to automate foundational security checks, including password strength analysis and basic system verification.
*   **Focus:** Python scripting, logic building, and security automation.

### 📂 Task 2: Vulnerability Assessment & Threat Modeling
*   **Description:** Conducted setup and architecture reviews across different operating systems (Linux, macOS, Windows) and generated sample vulnerability reports.
*   **Focus:** System administration, infrastructure security, and VAPT reporting.

### 📂 Task 3: AI-Powered Phishing Email Detection 
*   **Description:** Engineered a Machine Learning model using Python to parse, analyze, and classify emails as safe or phishing based on textual patterns.
*   **Focus:** Natural Language Processing (NLP), Data Parsing (CSV), and Predictive Defense Modeling.

### 📂 Task 4: Secure Login System WebApp
*   **Description:** Built a secure web application prototype focusing on safe authentication practices, proper routing, and defense against common web vulnerabilities.
*   **Focus:** Web Application Security, Secure Authentication, and Full-Stack Integration.
